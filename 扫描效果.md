最简单、效果也最像“扫描”的做法是：在图片上盖一层绝对定位的 `::after` 或 `div`，用 `linear-gradient` 做一条“扫描光带”，然后用 `keyframes` 让它从上到下、再回到上（来回扫描）。

下面给你 3 种方案（都不需要外部文件），你按你的网站技术栈选一个。

---

## 方案 A：纯 CSS 叠加扫描光带（推荐，最轻）

适用：你图片是 `<img>` 或背景图都行。

```html
<div class="scan-wrap">
  <img class="scan-img" src="/your-image.jpg" alt="preview" />
  <div class="scan-line"></div>
</div>
```

```css
.scan-wrap{
  position: relative;
  display: inline-block;
  overflow: hidden;
  border-radius: 12px; /* 可选 */
}

.scan-img{
  display:block;
  width: 360px;       /* 按需 */
  height: auto;
}

.scan-line{
  position:absolute;
  left:0; right:0;
  top:-30%;
  height:30%;
  pointer-events:none;

  /* 扫描“光带” */
  background: linear-gradient(
    to bottom,
    rgba(0,0,0,0) 0%,
    rgba(0,255,200,.10) 40%,
    rgba(0,255,200,.35) 50%,
    rgba(0,255,200,.10) 60%,
    rgba(0,0,0,0) 100%
  );

  /* 让光带更像扫过：轻微模糊 + 叠加模式 */
  filter: blur(0.5px);
  mix-blend-mode: screen;

  animation: scanY 1.6s ease-in-out infinite alternate;
}

@keyframes scanY{
  from { transform: translateY(0%); }
  to   { transform: translateY(430%); } /* 让光带从上扫到下，按图高调 */
}
```

要点：

* `alternate` 就是“来回扫”（下去再上来），不需要写两段关键帧。
* `height: 30%` 控制光带宽度。
* `mix-blend-mode: screen` 让它像“光”，如果你觉得太花，可以删掉。

---

## 方案 B：更像“扫描仪”的效果：光带 + 细网格线

在 A 的基础上，再加一层“细条纹”，会更有科技感：

```css
.scan-wrap::after{
  content:"";
  position:absolute;
  inset:0;
  pointer-events:none;

  /* 细网格/扫描纹理 */
  background: repeating-linear-gradient(
    to bottom,
    rgba(0,255,200,.08),
    rgba(0,255,200,.08) 1px,
    rgba(0,0,0,0) 3px,
    rgba(0,0,0,0) 6px
  );

  opacity:.18;
}
```

这层不动，只是加“扫描纹理”，动的还是 `.scan-line`。

## 方案 C：React + Tailwind（不写 CSS 文件）

如果你是 Next.js + Tailwind，用 `style jsx` 或 Tailwind 的 `@layer` 都行。这里给一个最直接的 React 组件（无外部文件）：

```jsx
export default function ScanningImage({ src }) {
  return (
    <div className="relative inline-block overflow-hidden rounded-xl">
      <img src={src} alt="preview" className="block w-[360px] h-auto" />
      <div className="pointer-events-none absolute inset-x-0 -top-[30%] h-[30%] scan-line" />

      <style jsx>{`
        .scan-line{
          background: linear-gradient(
            to bottom,
            rgba(0,0,0,0) 0%,
            rgba(0,255,200,.10) 40%,
            rgba(0,255,200,.35) 50%,
            rgba(0,255,200,.10) 60%,
            rgba(0,0,0,0) 100%
          );
          filter: blur(0.5px);
          mix-blend-mode: screen;
          animation: scanY 1.6s ease-in-out infinite alternate;
        }
        @keyframes scanY{
          from { transform: translateY(0%); }
          to { transform: translateY(430%); }
        }
      `}</style>
    </div>
  );
}
```

---

## 建议你顺手加两个“体验增强”

1. **只在加载/处理中显示扫描动画**
   处理完成就隐藏，避免一直晃。

2. **尊重系统“减少动画”设置**（更专业）

```css
@media (prefers-reduced-motion: reduce){
  .scan-line{ animation: none; }
}
```
